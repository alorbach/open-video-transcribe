## Screenshot Guidelines

- **Format**: PNG format recommended
- **Size**: 800-1200 pixels wide is ideal
- **Quality**: Clear, readable text and UI elements
- **Content**: Show relevant UI elements clearly
- **Privacy**: Ensure no personal information is visible

## How to Take Screenshots

### Windows
- Use Snipping Tool or Win+Shift+S
- Capture the relevant window/dialog
- Save as PNG

### Linux
- Use Screenshot tool (usually in Applications menu)
- Or use: `gnome-screenshot -w` (GNOME) or `spectacle` (KDE)

### macOS
- Use Cmd+Shift+4 for area selection
- Or Cmd+Shift+3 for full screen
- Save as PNG
